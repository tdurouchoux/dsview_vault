The Sufficient Context Autorater is a model designed to classify instances based on whether the provided context contains enough information to adequately answer a given query. It evaluates the relevance of the context in relation to the query without needing to reference a ground truth answer. This tool is crucial in analyzing model performance within retrieval augmented systems and can identify when a query can or cannot be answered based on the context provided.
