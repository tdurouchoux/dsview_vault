ModernBERT is a new family of encoder-only models designed as a replacement for BERT. It features improvements across various performance metrics, such as allowing for longer sequence lengths (8192 tokens), enhanced downstream performance, and faster processing. Unlike previous models, ModernBERT is tailored for efficiency and includes a diverse dataset for training, facilitating applications in code retrieval and natural language understanding.
