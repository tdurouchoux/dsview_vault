k-Means Clustering is a popular unsupervised learning algorithm used to partition a dataset into k distinct groups based on feature similarity. The algorithm initializes k centroids and iteratively assigns each data point to the nearest centroid and recalculates the centroids based on the current assignments. This process continues until the assignments no longer change. k-Means is widely used for applications such as market segmentation, image compression, exploratory data analysis, and as a preprocessing step for other algorithms.
