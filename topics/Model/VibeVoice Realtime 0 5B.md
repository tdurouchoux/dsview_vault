VibeVoice-Realtime-0.5B is a lightweight, real-time Text-to-Speech (TTS) model designed for streaming text input and robust long-form speech generation. With a parameter size of 0.5 billion, it is optimized for deployment-friendly applications. The model supports real-time TTS with a first audible latency of approximately 300 milliseconds and can generate speech for up to 10 minutes in a single stream.
