DeepEval is an open-source library designed for evaluating and testing the outputs of large language models (LLMs). It provides a framework similar to unit testing where metrics are defined to assess aspects such as correctness, hallucination, and relevance of LLM outputs. The library supports the integration of various evaluation metrics and can run on various environments, making it flexible for researchers and developers working with LLMs.
