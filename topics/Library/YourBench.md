YourBench is an open-source framework designed for dynamic, domain-specific benchmark generation for evaluating large language models (LLMs) against real-world datasets. It automates the creation of evaluation sets by processing various document formats and generating question-answer pairs without the need for manual labeling. This capability addresses evaluation challenges in rapidly evolving domains, thereby enhancing the relevance and accuracy of assessments.
