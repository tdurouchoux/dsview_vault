Pi packages are modular extensions, skills, prompt templates, or themes that can be installed in the Pi coding agent to customize its functionality. These packages can be shared via platforms like npm or GitHub, allowing users to extend Pi's capabilities without modifying the core system. Examples include tools for specific workflows, custom providers or models, or UI tweaks. Pi packages enable users to adapt the tool to their specific needs and share their customizations with others.
