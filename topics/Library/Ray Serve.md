Ray Serve is a scalable and programmable model serving library designed for building online inference APIs. It is framework-agnostic, meaning it supports serving models built with various frameworks like PyTorch, TensorFlow, Keras, Scikit-Learn, and even arbitrary Python business logic. Key features include response streaming, dynamic request batching, multi-node/multi-GPU serving, model composition, and resource flexibility such as fractional GPU allocation. Ray Serve is built on top of Ray, enabling it to scale across multiple machines and integrate seamlessly with other Ray libraries like Ray Data, Ray Train, and Ray Tune.
