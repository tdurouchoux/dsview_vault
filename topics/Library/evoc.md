EVōC (Embedding Vector Oriented Clustering) is a Python library designed for fast and flexible clustering of large datasets containing high-dimensional embedding vectors. It specializes in clustering embedding vectors generated by models like CLIP, sentence-transformers, OpenAI, or Cohere. EVōC improves upon traditional methods such as UMAP + HDBSCAN by optimizing for embedding vectors specifically, reducing the need for hyperparameter tuning and significantly speeding up the clustering process. Key features include support for multi-granularity clustering, automatic selection of the number of clusters, and the ability to cluster int8 or binary quantized embedding vectors out-of-the-box. The library follows the scikit-learn API, making it familiar and easy to integrate for users already accustomed to scikit-learn's clustering algorithms.
