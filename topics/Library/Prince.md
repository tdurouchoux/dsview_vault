Prince is a Python library designed for multivariate exploratory data analysis. It provides efficient implementations of various dimensionality reduction and data summarization techniques, including Principal Component Analysis (PCA), Correspondence Analysis (CA), Multiple Correspondence Analysis (MCA), Factor Analysis of Mixed Data (FAMD), Multiple Factor Analysis (MFA), Generalized Procrustes Analysis (GPA), and Principal Geodesic Analysis (PGA). The library follows a scikit-learn-compatible API, making it accessible and easy to integrate into Python-based data science workflows. Additionally, Prince uses Altair for data visualization.
