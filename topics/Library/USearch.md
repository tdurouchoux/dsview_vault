USearch is a lightweight, open-source library designed for fast similarity search and clustering of vectors and arbitrary objects. It supports a wide range of data types, including vectors, text, and other objects, and is optimized for performance across multiple programming languages such as C++, Python, JavaScript, Java, Rust, and more. USearch is designed to be extensible, allowing users to define custom metrics and integrate it with various applications like semantic search, molecular search, and geospatial information systems. It is hardware-agnostic, supporting SIMD optimizations and various precision formats (e.g., bf16, f16, i8) for efficient memory usage and performance.
