A Python library that provides bindings for the llama.cpp library, enabling low-level and high-level access to C API via ctypes interface. It supports text completion, OpenAI-like API compatibility, LangChain, and LlamaIndex integration. The library also offers features like function calling, vision API support, multi-modal model handling, speculative decoding, and embeddings. Additionally, it includes a web server for OpenAI-compatible API interactions.
