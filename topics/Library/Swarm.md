Swarm is an educational framework designed to explore ergonomic, lightweight multi-agent orchestration. It focuses on the coordination and execution of multiple agents, each capable of handling specific instructions and tools, which can interact and communicate effectively. This library provides abstractions like Agents for encapsulating instructions and handoffs for transferring control between agents, allowing for scalable and customizable solutions in multi-agent systems.
