TabPFN (Tabular Prior-data Fitted Networks) is a machine learning library and framework designed for tabular data prediction. It leverages Prior-Data Fitted Networks (PFNs) to enable efficient and accurate predictions, including in-context learning, without extensive hyperparameter tuning or feature engineering. TabPFN is pre-trained on synthetic and real data, allowing it to handle small to medium-sized tabular datasets with minimal computational overhead. It uses a transformer architecture to sample tasks and datasets, integrating over latent tasks to enable transfer learning across different tabular problems. It is widely recognized as a strong baseline for comparison in tabular data benchmarks.
