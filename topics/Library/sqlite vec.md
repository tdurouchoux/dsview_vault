sqlite-vec is a lightweight SQLite extension designed for vector searches. It allows storage and querying of floating-point, integer, and binary vectors using special virtual tables. The extension is built in pure C without dependencies, enabling it to run on various platforms including Linux, MacOS, Windows, and even in web browsers via WASM. It's particularly useful for data scientists needing efficient vector operations in SQLite databases.
