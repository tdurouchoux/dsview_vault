pytabkit is a Python library that provides implementations of state-of-the-art deep learning models specifically designed for tabular data. It offers optimized hyperparameter defaults, a unified API similar to scikit-learn, and automated preprocessing for heterogeneous data. pytabkit supports both classification and regression tasks, making it easier to experiment with modern tabular DL methods. The library is designed to address historical barriers to using tabular DL, such as implementation difficulty and the need for custom code.
