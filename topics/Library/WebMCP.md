WebMCP is an open-source JavaScript library designed to enable websites to integrate with the Model Context Protocol (MCP). It facilitates interaction between users and webpages through LLMs or agents by providing a small widget (typically a blue square) in the bottom right corner of the webpage. WebMCP allows developers to register tools, prompts, resources, and sampling functionalities, enabling dynamic and interactive experiences for users while maintaining security and privacy through human oversight.
