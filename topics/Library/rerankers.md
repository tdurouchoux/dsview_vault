rerankers is a lightweight Python library designed to provide a unified interface for various re-ranking methods used in information retrieval. It aims to simplify the integration of multiple re-ranking approaches, allowing data scientists to easily swap between them with minimal coding. The library is designed to be low-dependency, modular, and extensible, making it easy to add new re-ranking methods.
