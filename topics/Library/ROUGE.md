ROUGE, which stands for Recall-Oriented Understudy for Gisting Evaluation, is a set of metrics used for evaluating automatic summarization systems. It specifically measures the overlap of n-grams between generated summaries and reference texts, facilitating the assessment of content retention, coherence, and information preservation between documents. It is widely used in Natural Language Processing (NLP) tasks for its straightforward implementation and effectiveness in quantifying summarization quality.
