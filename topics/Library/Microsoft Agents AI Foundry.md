A software library developed by Microsoft as part of the Agent Framework. It provides tools and abstractions for building AI agents and workflows, enabling integration with services like Microsoft Foundry, Anthropic, Azure OpenAI, OpenAI, and Ollama. The library simplifies the creation of conversational AI applications, multi-agent systems, and graph-based workflows with features like state management, type safety, middleware, and human-in-the-loop support.
