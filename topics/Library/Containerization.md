Containerization is a Swift package that provides the functionality to run Linux containers on macOS. It leverages the Virtualization framework available on Apple silicon, allowing for streamlined management of OCI images, interaction with remote registries, and the ability to create containerized applications efficiently. The library offers optimized modifications to the Linux kernel to enhance performance, facilitating fast startup times for containers.
