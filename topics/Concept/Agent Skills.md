Agent Skills refer to the process of encapsulating repetitive or complex workflows, instructions, scripts, and resources into reusable and portable packages that AI models or agents can execute on demand. These skills provide procedural knowledge, context-specific information, and the ability to interact with the environment through scripting and CLIs, enabling agents to perform tasks more accurately, efficiently, and flexibly. They reduce the need for manual intervention, formalize organizational knowledge, and support consistent, auditable, and scalable workflows. Examples include skills for polishing code, writing drafts, generating task lists, or executing dynamic scripts to handle complex tasks.
