Secure Aggregation refers to techniques used in federated learning to combine the contributions of individual models from users in a secure manner. It ensures that the central server receives a global model without accessing the individual updates from user devices, thereby preserving privacy. This process often employs cryptographic methods to mask the individual contributions, allowing for the collaborative improvement of the model while maintaining confidentiality.
