Reservoir sampling is a randomized algorithm for selecting a fair sample of elements from a stream or collection without knowing its total size in advance. It ensures that all elements have an equal probability of being included in the final sample, which is crucial in situations where the dataset is large or infinite. The technique allows for efficient memory usage by retaining only a fixed number of sampled elements (k) at any one time during streaming data.
