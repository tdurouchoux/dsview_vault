Stealth mode refers to techniques and methods used to avoid detection by websites that aim to block automated tools like scrapers or bots. This includes masking browser fingerprints (e.g., GPU, screen resolution, canvas rendering), randomizing user agent strings, disabling flags like `navigator.webdriver`, blocking trackers, and employing modified browsers with fingerprint spoofing capabilities. Stealth mode enhances the ability of headless browsers to operate undetected, bypass anti-bot systems, and avoid being blocked during web scraping or automation tasks.
