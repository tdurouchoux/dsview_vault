An advanced paradigm in data analytics that leverages AI, particularly generative AI and large language models, to enable users to interact with data using natural language. This approach transforms traditional static dashboards into dynamic, interactive workspaces where users can query data, generate insights, create dashboards, and refine analytics artifacts in real time without requiring technical expertise. It combines conversational analytics, which focuses on natural language interactions for data querying and insight generation, with Generative Business Intelligence (GenBI), which extends these capabilities to include the creation and iteration of analytics artifacts like dashboards and metrics through declarative BI stacks.
