Zero-trust backup is a security model where backups are encrypted before leaving the user's system, ensuring that even the storage provider cannot access the plaintext data. This approach relies on encryption keys stored in the user's own secret manager, making backups tamperproof and immutable (especially when using object-lock storage). It ensures that only authorized users with the correct keys can access or restore the data, mitigating risks like ransomware and rogue operators.
