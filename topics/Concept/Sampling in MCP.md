Sampling in the context of the Model Context Protocol (MCP) refers to the process where an LLM requests a human user to provide input or approval for a specific action or response. This is typically done through a modal dialog displayed by the MCP Client (e.g., WebMCP widget), ensuring that human oversight is maintained for security and privacy. Sampling is used to gather user input for tasks like generating responses, approving actions, or providing additional context.
