A metadata graph is a structured representation of metadata and its relationships, often visualized as a network of nodes (e.g., datasets, dashboards, ML models) and edges (e.g., lineage, dependencies). Tools like DataHub use metadata graphs to provide a unified view of an organization's data ecosystem, enabling efficient discovery, governance, and impact analysis. The graph is typically stored and queried using graph databases or graph-based APIs.
