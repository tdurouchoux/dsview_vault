Token efficiency refers to the ability of a model to achieve high task accuracy while consuming fewer tokens. Models vary substantially in their token efficiency, with some models like GPT-5 and GPT-5.2 consuming significantly fewer tokens than others (e.g., Kimi-K2 and Claude-Sonnet-4.5) for the same tasks. Token efficiency is an inherent characteristic of the model and reflects its behavioral tendencies during task execution.
