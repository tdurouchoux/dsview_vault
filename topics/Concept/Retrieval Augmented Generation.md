Retrieval Augmented Generation (RAG) is a hybrid approach that combines retrieval systems with generative models to enhance performance in tasks like open-domain question answering, language generation, and code completion. By fetching relevant context from external knowledge bases during inference, RAG systems improve the factual accuracy, relevance of generated answers, and reduce instances of hallucination, grounding outputs in real-time data.
