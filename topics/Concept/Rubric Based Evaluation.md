Rubric-based evaluation is a method of assessing code quality or performance using a predefined set of criteria (a rubric) that are weighted and categorized as either blockers or non-blockers. Blocker criteria represent hard requirements for mergeability, such as correctness and regression safety, while non-blocker criteria assess softer qualities like code style and readability. This approach allows for a nuanced evaluation that reflects real-world coding standards and maintainer expectations.
