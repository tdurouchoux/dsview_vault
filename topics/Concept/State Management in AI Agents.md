A system for maintaining and managing the state of AI agents and workflows over time, including agent sessions that retain persistent state across interactions. This encompasses tracking conversation history, session data, container state, file system changes, and intermediate results, enabling seamless resumption of work, context retention, and continuity in task execution. It is critical for long-running processes, human-in-the-loop scenarios, and core design principles in AI agent systems like Claude Managed Agents.
