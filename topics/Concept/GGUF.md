GGUF (GGML Universal Format) is a binary file format and framework designed for storing, distributing, and optimizing the execution of quantized large language models (LLMs). It includes model weights, metadata, and tokenizer information, and is optimized for efficient loading and inference. GGUF is particularly suited for resource-constrained environments such as browsers or edge devices, offering smaller file sizes and faster loading times compared to traditional model formats. It supports various quantization techniques and is compatible with frameworks like GGML and applications such as Ollama.
