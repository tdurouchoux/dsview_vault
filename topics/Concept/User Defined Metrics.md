User-Defined Metrics refer to the ability to customize the distance or similarity function used in vector search operations. This flexibility allows users to tailor the search process to specific applications, such as geospatial calculations (e.g., Haversine distance) or domain-specific embeddings (e.g., composite image-text embeddings). USearch supports arbitrary user-defined metrics, enabling integration with external libraries like Numba for metric computation in Python.
