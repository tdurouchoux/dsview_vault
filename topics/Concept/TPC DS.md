TPC-DS (Transaction Processing Performance Council Decision Support) is a standardized benchmark used to evaluate the performance of decision support systems, including data warehouses and big data processing engines. It simulates complex, real-world queries and workloads to measure system efficiency, scalability, and throughput. The benchmark models several generally applicable aspects of a decision support system, including queries and data maintenance, providing a representative evaluation of performance in both single-user and multi-user modes. It measures query response time, query throughput, and data maintenance performance for a given hardware, operating system, and data processing system configuration. TPC-DS is widely used to benchmark tools like Apache DataFusion Comet and traditional systems like Apache Spark, enabling emerging technologies, such as Big Data systems, to execute the benchmark. The TPC-DS Price/Performance metric is expressed as Price/QphDS@Size for Version 2 and Price/kQphDS@Size for Version 3.
