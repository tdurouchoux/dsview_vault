Agent instruction files, such as AGENTS.md, are standardized documents within a codebase designed to provide guidance to AI coding agents. These markdown-based files, like AGENTS.md, serve as a universal standard for offering project-specific instructions, context, or references to additional resources (e.g., wikis). Developed by organizations like OpenAI and adopted by numerous open-source projects, these files enhance the predictability of agent behavior, enabling agents to interact more effectively with repositories and retrieve relevant context dynamically.
