Joint-Embedding Predictive Architecture (JEPA) is a framework for self-supervised learning that focuses on predicting representations of target data blocks from a given context block. Unlike traditional generative models that predict raw data (e.g., pixels), JEPA operates in representation space, enabling it to learn more semantic and abstract features. This approach avoids the need for hand-crafted data augmentations and is designed to capture high-level semantic information, making it versatile across different modalities such as images, text, and audio.
