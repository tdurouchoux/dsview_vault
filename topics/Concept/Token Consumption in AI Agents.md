Token consumption refers to the usage of input and output tokens by AI agents during task execution. Input tokens dominate the cost in agentic coding tasks due to the accumulation of context from multiple rounds of interaction, tool calls, and file inspections. The variability in token usage is inherently stochastic, with runs on the same task differing significantly in total tokens consumed. This concept is critical for understanding the economics of AI agents and optimizing their deployment.
