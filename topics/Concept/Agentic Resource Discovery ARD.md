Agentic Resource Discovery (ARD) is an open discovery protocol that enables AI clients or agents to dynamically query and identify external resources, tools, skills, or other agents across decentralized or federated registries. It focuses on the discovery phase, allowing agents to find and integrate capabilities at runtime based on intent-based queries, rather than relying on pre-installed or manually configured catalogs. ARD includes a static manifest format (ai-catalog.json) for publishers to host their capabilities and a dynamic registry API (POST /search) for live discovery, aiming to scale agentic systems by reducing manual integration efforts and fostering access to a broader ecosystem of tools and services.
