The FACTS Grounding Dataset is a specialized collection of examples designed to assess the performance of large language models in generating factually accurate, long-form responses grounded in provided documents. This dataset consists of 1,719 examples that require models to respond comprehensively to user requests, ensuring that answers can be traced back to specific content within documents. It includes a public set for evaluation and a private set to prevent benchmark contamination.
