Graph-based metrics serve as evaluation strategies for assessing the performance of prompting techniques used with large language models. By modeling thoughts as graphs, various metrics can be developed to evaluate properties such as the volume of a thought, which indicates the number of preceding thoughts that can influence a particular output. This quantitative approach aims to provide insights into the effectiveness of different prompting schemes, guiding towards more optimized designs.
