The DE1-SoC is a development board featuring an Intel Cyclone V FPGA, used as the target platform for the Talos V2 project. The Cyclone V FPGA provides the hardware resources (e.g., logic blocks, memory, and DSP units) required to implement the RTL design. The board's constraints, such as memory layout, clocking, and resource availability, shape key design decisions in Talos V2, including tile width, clocking strategy, and memory usage patterns.
