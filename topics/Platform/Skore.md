Skore is a collaboration layer for data science teams, designed to help data scientists track and manage their work. It integrates with scikit-learn and provides tools for model evaluation, cross-validation, benchmarking, and collaboration. Skore aims to standardize workflows, reduce boilerplate code, and guide users toward methodological best practices. It is built to be composable, reusable, and transparent, fitting into existing data science stacks without creating vendor lock-in.
