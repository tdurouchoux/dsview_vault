Rill is a BI tool that applies the philosophy of BI-as-Code, allowing users to define dashboards and metrics in a declarative code format. Rill utilizes Generative AI to assist in creating and modifying dashboard definitions directly from coded specifications. This approach enhances user engagement with data through conversational AI interfaces, making data visualization and analysis more accessible.
