TabArena is a benchmarking platform and leaderboard for evaluating machine learning models on tabular data. It provides standardized datasets, consistent evaluation metrics, and transparency in preprocessing and hyperparameter choices. TabArena addresses the issue of unfair comparisons in previous studies by offering a curated set of tabular datasets for binary/multiclass classification and regression tasks. It also uses Elo-style model rankings to facilitate easier comparisons between models.
