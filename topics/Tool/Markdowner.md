Markdowner is a tool designed to convert website content into structured markdown format, optimized for use with Large Language Models (LLMs). It supports features like LLM filtering, detailed markdown mode, auto-crawling of subpages, and the ability to return responses in text or JSON format. The tool leverages Cloudflare's Browser Rendering and Durable Objects to spin up browser instances and convert HTML content to markdown using Turndown. It can be self-hosted and is built to address issues of cost, limitations, and deployment complexity in existing solutions.
