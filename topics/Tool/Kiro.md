Kiro is a lightweight tool for spec-driven development that operates within a VS Code environment. It guides users through a structured workflow consisting of three main steps: Requirements, Design, and Tasks. Each step is documented in markdown files, and Kiro provides UI elements to manage and review tasks. Kiro also includes a concept called 'steering,' which acts as a memory bank containing high-level context documents relevant across all AI coding sessions in the codebase.
