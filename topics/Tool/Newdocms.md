Newdocms is an innovative document storage and retrieval system designed to transcend the limitations of traditional hierarchical file systems. It allows users to save documents using a flat storage model while organizing them through a rich set of metadata attributes represented as (attribute,value) pairs. Users can define relationships between documents and concepts, facilitating more intelligent and flexible organization and retrieval through a category tree structure.
