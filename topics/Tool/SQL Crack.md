SQL Crack is a Visual Studio Code (VS Code) extension designed to visualize SQL queries as interactive execution flow diagrams. It helps users understand complex SQL queries by breaking them down into step-by-step, color-coded operation nodes. Key features include multi-query support, column lineage tracing, CTE and subquery expansion, query comparison, and performance analysis with anti-pattern detection. It also supports workspace analysis for cross-file dependencies, data lineage exploration, and impact analysis. SQL Crack aims to enhance SQL query comprehension, optimization, and debugging by providing a visual and interactive representation of SQL execution flows and dependencies.
