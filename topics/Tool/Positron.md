Positron is a next-generation data science Integrated Development Environment (IDE) developed by Posit PBC. It is designed to support code writing and data exploration in an extensible and polyglot environment, making it suitable for a wide range of data science applications. As an early-stage project, it promotes feedback from users and is built leveraging technologies from Visual Studio Code, including features for source control and customizable settings.
