Web Search and Fetch Tools are utilities that enable models to retrieve real-time information from the internet. Web Search allows models to query external sources for relevant data, while Web Fetch enables them to access and extract content from specific URLs. These tools are critical for tasks requiring up-to-date or domain-specific information, such as deep research or fact-checking. However, they must be used carefully to avoid benchmark contamination or reliance on unverified sources.
