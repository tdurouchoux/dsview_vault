Datanomy is a terminal-based tool designed for inspecting and understanding data files, specifically Parquet files. It provides an interactive view of the data's structure, metadata, and internal organization. Datanomy allows users to explore the general structure, schema, data, metadata, and statistics of Parquet files. It is built with Textual and PyArrow and can be installed via PyPI or from the source. The tool supports various features such as keyboard shortcuts for navigation and interaction within the terminal.
