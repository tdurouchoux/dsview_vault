InstructPipe is a prototype tool designed to assist users in generating machine learning pipelines through interactive visual programming. It leverages large language models (LLMs) to interpret natural language instructions and automate the selection and connection of nodes within a visual programming environment. By lowering the barrier to creating complex pipelines, InstructPipe empowers users, especially novices, to prototype and explore machine learning workflows effectively.
