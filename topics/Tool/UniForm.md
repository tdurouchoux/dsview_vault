UniForm is an open-source technology developed to enable cross-format compatibility for data lakehouse tables. It allows tables stored in one format (e.g., Delta Lake) to be read and queried by engines designed for other formats (e.g., Apache Iceberg or Apache Hive) without requiring data duplication. UniForm simplifies data management by ensuring that tables remain accessible across different compute engines, enhancing interoperability and reducing vendor lock-in.
