Surya is an OCR (Optical Character Recognition) toolkit that supports reading and processing text from images and documents in over 90 languages. It provides functionalities such as line-level text detection, layout analysis, reading order detection, table recognition, and LaTeX OCR. It is designed to work on various types of documents including scans and PDFs and is optimized for performance using CPU or GPU.
