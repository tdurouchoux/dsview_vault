Mistral Vibe CLI is an open-source command-line coding assistant powered by Devstral. It provides an interactive chat interface with tools for file manipulation, code searching, version control, and command execution. Key features include project-aware context, smart references, multi-file orchestration, persistent history, autocompletion, and customizable themes. It can be integrated into preferred IDEs via the Agent Communication Protocol and is released under the Apache 2.0 license.
