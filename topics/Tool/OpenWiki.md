OpenWiki is an open-source tool designed to autonomously generate and maintain documentation for codebases. It creates a structured wiki that helps coding agents understand the repository they are working in by providing context about key logic, file connections, and codebase patterns. OpenWiki automates the process of updating documentation as the code changes, ensuring that the wiki remains current without manual intervention. It integrates with coding agents by updating agent instruction files (e.g., AGENTS.md or CLAUDE.md) to reference the generated wiki, allowing agents to retrieve relevant context dynamically.
