Apache Spark is an open-source distributed computing system that serves as a unified analytics engine for big data processing. It includes high-level APIs in Java, Scala, Python, and R, and supports various modules for streaming, SQL with Spark SQL, machine learning through MLlib, and graph processing. Spark is known for its ability to process large-scale data sets across clusters.
