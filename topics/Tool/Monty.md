Monty is a minimal, secure Python interpreter written in Rust, designed specifically for executing code generated by AI agents. It provides a lightweight and fast alternative to traditional sandboxing methods like Docker containers, enabling safe execution of Python code with controlled access to system resources. Monty supports a subset of Python's standard library, modern type hints, and can be integrated with external functions. It also offers features like snapshotting, serialization, and resource usage control, making it suitable for use in agentic workflows where security and performance are critical.
