Plakar is an open-source, efficient, and secure backup and restore tool designed for modern infrastructure. It supports deduplication, compression, and encryption of data before it leaves the user's system, ensuring minimal storage and egress overhead. Plakar is platform-agnostic, supporting on-premises, cloud (AWS, GCP, Azure, OVHCloud, etc.), and hybrid environments. It integrates with various data sources like Kubernetes, PostgreSQL, MySQL, VMware vSphere, and Docker, and provides features such as immutable backups (via object-lock), zero-trust security, and programmatic control via SDK, CLI, and API. Plakar also supports 'Resilience as Code,' allowing backup policies to be defined, versioned, and automated using Kubernetes CRDs, Terraform, or Ansible.
