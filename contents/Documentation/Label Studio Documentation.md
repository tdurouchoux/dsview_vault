## Summary

Label Studio 1.15.0 introduces new tools for PDF labeling. The documentation covers various aspects, including installation, project management, data import/export, and machine learning integration. Key sections include:

- **Installation & Setup**: Guides for quick start, Kubernetes installation, database setup, and securing the application.
- **Project Management**: Instructions for creating and managing projects, configuring labeling interfaces, and utilizing data management features.
- **Import & Export**: Details on adding project storage, importing data (including pre-annotations), and exporting annotations.
- **Machine Learning**: Information on integrating ML models, writing custom ML backends, and accessing examples and tutorials.
- **Integration & Extension**: API usage, Python SDK, and webhook development.

Additional resources include a quick start guide, video tutorials, and community support options.
