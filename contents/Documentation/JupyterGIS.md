## Summary

JupyterGIS is a collaborative GIS environment within JupyterLab, enabling real-time teamwork on geospatial projects. Key highlights include simultaneous multi-user editing (like Google Docs), basic QGIS project import/export support, and a Python API integrated with Jupyter workflows. It’s part of the open GeoJupyter community, aiming to democratize geospatial data access. The documentation covers an overview, user guide (installation, tutorials, API), and contributor guide (setup, troubleshooting). Try it via JupyterLite (local-only, no collaboration).
