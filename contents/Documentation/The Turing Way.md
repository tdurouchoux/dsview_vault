## Summary

The Turing Way is an open, community-driven guide for reproducible, ethical, and collaborative data science. It provides resources for researchers, data scientists, and policymakers to ensure projects are reusable and transparent. The handbook is modular—users can explore specific guides (e.g., reproducible research, project design, communication) or search for topics as needed. It’s collaboratively written, with contributions from global experts across academia, industry, and government. The project emphasizes inclusivity, offering translation opportunities and a code of conduct. Licensed under CC-BY 4.0, all materials are freely reusable. Community engagement is encouraged via GitHub, Slack, and newsletters. The Foreword section outlines navigation, citation, and involvement guidelines.
