## Summary

Pointblank is a Python library designed for table validation and testing, ensuring that tabular data meets specified expectations and constraints. It generates detailed validation reports and supports various data sources, including Polars and Pandas DataFrames, as well as databases like MySQL and PostgreSQL.

Key features include:
- Flexibility in handling multiple table formats.
- Generation of visually appealing HTML reports.
- Functional outputs for further data processing.
- An easy-to-use API with clear documentation.

To get started, users can install Pointblank via pip and utilize the `Validate` class to create validation plans with various steps, such as checking column values and existence. The results can be viewed in a tabular format, showing pass/fail statuses for each validation step.

The library is open for contributions, with a roadmap outlining future enhancements, including more validation methods, logging functionalities, and CLI utilities. The project adheres to a code of conduct and is maintained by Rich Iannone, with contributions welcomed from the community.
