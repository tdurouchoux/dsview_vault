## Summary

skrub is a Python library designed for preprocessing and feature engineering of tabular data, compatible with scikit-learn and capable of handling heterogeneous dataframes (numerical, categorical, dates, text) from sources like databases. 

Key features include:

- **Effortless Pipelines**: Create machine learning pipelines easily using `TableVectorizer` and `tabular_learner`.
- **Feature Engineering**: Encode text and high cardinality categorical data using `GapEncoder` and `MinHashEncoder`, and extract features from dates with `DatetimeEncoder`.
- **Interactive Data Exploration**: Explore dataframes with `TableReport`.

The library aims to streamline the connection between database tables and machine learning estimators, with a community-driven approach for ongoing development and enhancement.
