## Summary

FollowTheMoney (FtM) is a data model designed for anti-corruption investigations, providing definitions for entities like people and companies. It allows users to generate, validate, and export data, facilitating relationship mapping through entity references. FtM can be utilized as a command-line tool, a Python library, or a TypeScript/JavaScript library. The model also encompasses various document types that may serve as evidence in investigations. Additionally, the Aleph search engine utilizes FtM entities, offering enhanced functionalities for searching, viewing, and managing these entities, alongside higher-level constructs for datasets and access control.
