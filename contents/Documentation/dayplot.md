## Summary

dayplot is a Python library designed for creating calendar heatmaps with ease, built on top of Matplotlib for high customization. Key functionalities include:

- **Installation**: Install via pip (`pip install dayplot`) or from GitHub.
  
- **Basic Usage**: Users can load datasets using `dp.load_dataset()` and create calendar heatmaps with options for dates, values, and customization attributes like color maps and box styles.

- **Custom Styling**: The library offers basic styling, support for negative values, and various box styles (e.g., circle).

- **Related Projects**: Includes tools like calplot and july for additional calendar visualization options.

Overall, dayplot streamlines the process of visualizing time-series data within a calendar framework.
