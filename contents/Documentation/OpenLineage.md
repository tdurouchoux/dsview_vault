https://openlineage.io/
## Summary

OpenLineage is an open framework for data lineage collection and analysis. It standardizes metadata collection about datasets, jobs, and runs to help users understand data production, usage, and impact of changes. Core components include:

- **Open standard API** for capturing lineage events (runs, jobs, datasets).
- **Marquez** as a reference metadata repository implementation.
- **Libraries** for common languages and **integrations** with data pipeline tools.

Supports simple and complex deployments. Open to community contributions via Slack, GitHub, and monthly TSC meetings. Focuses on enabling context-aware data tools and best practices.
