## Summary

Verba is an open-source Retrieval Augmented Generation (RAG) chatbot powered by Weaviate, designed for easy data querying and interaction. It supports various deployment options, including local installation, building from source, and Docker deployment. Key features include:

- **Model and Embedding Support**: Compatible with models from Ollama, HuggingFace, Cohere, Anthropic, and OpenAI.
- **Data Ingestion**: Supports multiple data formats such as PDFs, CSVs, and documents from GitHub/GitLab.
- **RAG Features**: Includes hybrid search, autocomplete suggestions, filtering, and customizable metadata.
- **Chunking Techniques**: Offers various methods for data chunking, including by token, sentence, and semantic similarity.
- **Community Contribution**: Encourages open-source contributions and provides a platform for community support.

To get started, users can install Verba via pip, build from source, or deploy using Docker, with detailed instructions provided for each method. The application allows users to import data, query it, and receive insights based on their chosen models.
