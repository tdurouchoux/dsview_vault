## Summary

LlamaParse is a GenAI-native document parser designed for optimal retrieval-augmented generation (RAG) applications. Key features include:

- **Broad File Type Support**: Handles various unstructured file formats such as PDF, PPTX, DOCX, XLSX, and HTML.
- **Table Recognition**: Accurately parses embedded tables into text and semi-structured formats.
- **Multimodal Parsing**: Extracts visual elements (images/diagrams) and returns them in structured formats.
- **Custom Parsing**: Allows users to input custom prompt instructions for tailored output.

LlamaParse integrates with LlamaIndex and offers a free plan for up to 1000 pages daily, with paid options available. The setup involves obtaining an API key, installing the necessary packages, and using command-line or script-based methods to parse documents. The documentation provides detailed guidance on usage and integration with other tools.
