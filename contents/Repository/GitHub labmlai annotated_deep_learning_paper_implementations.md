## Summary

The repository contains over 60 implementations and tutorials of deep learning papers, complete with side-by-side notes for better understanding. Key areas covered include:

- **Transformers**: Various types including original, XL, and ViT.
- **Optimizers**: Implementations like Adam, AdaBelief, and others.
- **Generative Adversarial Networks (GANs)**: Including CycleGAN and StyleGAN2.
- **Reinforcement Learning**: Techniques like PPO and DQN.
- **Capsule Networks, Distillation, and various normalization layers**.
- **Sampling techniques and scalable training/inference methods**.

The implementations are primarily done in PyTorch and aim to help users grasp complex algorithms effectively.
