## Summary

Latent Scope is a tool designed to explore and visualize datasets through latent spaces, particularly focusing on embedding unstructured data into high-dimensional vectors. It simplifies the workflow of embedding, dimensionality reduction, clustering, and labeling with a web interface and interactive visualizations.

Key features:
- Supports embedding of unstructured data followed by dimensionality reduction using UMAP and clustering with HDBSCAN.
- Offers auto-labeling of clusters using language models (LLMs).
- Datasets are stored in flat files for easy export and management.
- Provides a command line interface (CLI) alongside a web interface for flexibility.

Getting started involves installing the Python module, setting up a virtual environment, and running commands to ingest data, embed it, apply UMAP, cluster, label, and eventually serve the data for exploration. The tool is compatible with Mac, Linux, and Windows, and is geared for experimentation in both research and practical data science workflows.
