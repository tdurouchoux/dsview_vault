## Summary

The GitHub repository bytedance/coconut_cvpr2024 presents COCONut, a large-scale human-verified dataset aimed at enhancing segmentation tasks in computer vision. Key highlights include:

- COCONut is the first large-scale dataset with exquisite annotations, available for download on Kaggle and Hugging Face.
- It features multiple dataset splits: COCONut-S (118K images, 1.54M masks), COCONut-B (242K images, 2.78M masks), and COCONut-L (358K images, 4.75M masks), along with relabeled COCO-Val and COCONut-val.
- Tutorials are provided for various segmentation tasks, including semantic, instance, and panoptic segmentation, as well as open-vocabulary segmentation and object detection.
- The dataset is intended for research purposes and follows the same licensing as the COCO dataset, prohibiting commercial use.
- Users can download annotations via provided scripts and are encouraged to upvote the dataset on Kaggle if found useful.

For further details, users are directed to the FAQ and tutorials available in the repository.
