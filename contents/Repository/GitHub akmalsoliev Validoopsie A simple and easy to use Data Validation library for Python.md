## Summary

Validoopsie is a lightweight and user-friendly data validation library for Python, allowing developers to easily declare classes and chain validations for DataFrames. It utilizes a "Bring Your Own DataFrame" (BYOD) approach, making it compatible with various DataFrame types. The library's syntax emphasizes simplicity and readability, enabling straightforward data integrity checks and compliance validation.

Key features include:
- Easy installation via pip.
- A straightforward validation chaining mechanism using the `Validate` class.
- Validation methods encapsulated in distinct functions for clarity.
- Ability to handle high-impact errors and provide detailed validation results, while using loguru for colored output.

The library is inspired by Great Expectations and aims to streamline validation processes. Development aids like a Makefile for setup, linting, and testing are also provided, along with detailed documentation. Validoopsie is open-sourced under the MIT license.
