## Summary

Local Deep Research is an AI-powered assistant designed to convert complex queries into detailed, cited reports. It utilizes any large language model (LLM) to perform iterative analyses across various knowledge sources, including academic databases and private documents.

Key features include:

- **Advanced Research Capabilities**: Automated deep research, proper citations, and comprehensive webpage analysis.
- **Flexible LLM Support**: Supports local AI processing and cloud models, allowing for customizable model selection.
- **Rich Output Options**: Generates detailed reports, quick summaries, and tracks sources.
- **Privacy-Focused**: Operates locally for enhanced data security.
- **Enhanced Search Integration**: Utilizes various search engines intelligently and allows local document searches.

The system allows users to set up different configurations for AI models and search engines, with support for various local and cloud providers. It features both a web interface and command-line functionality for user interaction and report generation. The project emphasizes collaborative contributions and provides guidelines for developers.
