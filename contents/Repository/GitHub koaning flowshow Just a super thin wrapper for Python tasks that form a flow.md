## Summary

Flowshow is a lightweight Python library designed to facilitate the creation and management of task executions in a flow format. Key features include:

- A `@task` decorator to define tasks that can be tracked and visualized.
- A context manager, `span`, used for tracking the execution context within tasks.
- Logging utilities such as `info`, `warning`, and `add_artifacts` to enhance task documentation.

To use Flowshow, you install it via `pip install flowshow`, and leverage the decorators and context managers to create and monitor task flows. It supports retry logic on function failures and integrates smoothly with interactive notebooks, allowing for insightful visual representations of task executions.
