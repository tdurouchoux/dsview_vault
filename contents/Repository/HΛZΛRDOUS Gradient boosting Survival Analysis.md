## Summary

hazardous is a Python library designed for survival analysis and competing risks, featuring a scalable gradient-boosting model called SurvivalBoost. It is compatible with scikit-learn and provides evaluation metrics such as Integrated Brier Score, C-index, and Accuracy in Time.

Key distinctions include:
- **Survival Analysis**: Focuses on predicting the time until a single event occurs.
- **Competing Risks**: Involves multiple potential events, predicting which will occur first.

SurvivalBoost emphasizes predictive accuracy and integrates with scikit-learn tools for model training and evaluation. The library utilizes lifelines for certain estimators and metrics but does not currently support causal inference or statistical association quantification.

The library is under development, with an MIT license, and its API may change.
