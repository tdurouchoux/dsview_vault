## Summary

Data Formulator is an AI-powered tool developed by Microsoft Research that facilitates the creation of rich visualizations from data. It combines user interface interactions with natural language inputs, allowing users to describe chart designs while the AI handles data transformation. Key features include:

- **Installation Options**: Users can set it up via Python PIP, GitHub Codespaces, or in developer mode for customization.
- **Functionality**: Users provide OpenAI keys, select models, and choose datasets. They can specify chart types and data fields, and the AI generates visualizations based on these inputs.
- **Iterative Process**: Users can refine visualizations through follow-up prompts and track their exploration in the Data Threads panel.
- **Research and Contributions**: The project encourages contributions and has published research papers detailing its methodology.

The tool aims to streamline data visualization processes, making it accessible for analysts and developers alike.
