## Summary

Verba is an open-source Retrieval Augmented Generation (RAG) chatbot powered by Weaviate, designed for easy data querying and interaction. It supports various deployment options, including local, Docker, and cloud-based solutions. Key features include:

- **Model and Embedding Support**: Integrates with multiple LLM providers like OpenAI, Anthropic, and HuggingFace, allowing for local and cloud-based model usage.
- **Data Ingestion**: Supports various data formats including PDFs, CSVs, and audio files, with capabilities for scraping and importing from GitHub and GitLab.
- **RAG Features**: Offers hybrid search, autocomplete suggestions, customizable metadata, and advanced querying options.
- **Chunking Techniques**: Provides multiple methods for data chunking, including by token, sentence, and semantic similarity.
- **Deployment**: Can be installed via pip, built from source, or deployed using Docker, with detailed instructions for each method.

Verba encourages community contributions and provides a comprehensive FAQ section addressing common issues and deployment queries.
