## Summary

Latent Scope is a tool designed for exploring and visualizing datasets through latent spaces, leveraging machine learning to encode unstructured data into high-dimensional embeddings. It facilitates a workflow that includes embedding, projecting, clustering, and labeling data, allowing users to gain insights into their datasets.

Key features include:
- **Data Exploration**: Users can perform similarity searches, view automatically labeled clusters, and examine individual data points while maintaining context.
- **Web Interface**: A user-friendly web UI enables interaction with the processed data.
- **Command Line Interface (CLI)**: Users can also utilize a suite of command line scripts for data processing.
- **Integration with Python**: The tool supports data ingestion from Pandas dataframes and provides a Python interface for various operations.

The process involves several steps:
1. **Ingesting Data**: Converts input data into a structured format.
2. **Embedding**: Transforms text data into embeddings using various models.
3. **UMAP Projection**: Reduces dimensionality for visualization.
4. **Clustering**: Groups similar data points.
5. **Labeling**: Automatically labels clusters using language models.
6. **Serving**: Starts a server to explore the processed data.

Latent Scope is compatible with Mac, Linux, and Windows, and is built to support both local and API-based embedding models. The tool emphasizes ease of use, data exportability, and the ability to track experimental parameters for research purposes.
