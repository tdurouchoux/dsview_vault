## Summary

The smol_tools repository on GitHub offers a collection of lightweight AI-powered tools designed to run locally without requiring high-end GPU resources. Key features include:

- **SmolSummarizer**: Provides quick text summarization while maintaining key points and can respond to follow-up questions.
- **SmolRewriter**: Rewrites text to enhance professionalism and approachability while preserving the original intent.
- **SmolAgent**: An AI agent capable of performing various tasks, including weather lookups and web browsing, with an extensible tool system for additional capabilities.

Installation involves cloning the repository, setting up a virtual environment, and installing dependencies. A Tkinter-based demo application is available for user-friendly interaction, along with programmatic usage examples for integrating the tools into applications.

The tools utilize the SmolLM2-1.7B Instruct model, optimized for efficient inference. The project is licensed under the Apache License 2.0, and contributions are encouraged.
