## Summary

This repository provides the official implementation of the "Graph of Thoughts" (GoT) framework designed for solving complex problems using Large Language Models (LLMs). The core concept involves modeling problems as a Graph of Operations (GoO), enabling flexibility and extensibility in problem-solving methodologies, including architectures resembling previous approaches such as Chain of Thought (CoT).

Key components include:

- **Installation**: Requires Python 3.8 or newer, with options for standard or editable installation modes.
- **Configuration**: Users need access to an LLM and must configure it accordingly.
- **Implementation**: The framework allows the user to implement and run sorting problems using both CoT-like and GoT approaches through provided snippets.
- **Documentation**: Code and individual modules are extensively documented to aid understanding.
- **Examples**: The repository includes various examples that illustrate problem-solving capabilities within GoT, complete with guidance for execution.

Users can compare outputs from different methodologies and explore detailed results from the respective experiments outlined in the related paper. The repository encourages citation for academic use and provides contact details for feedback.
