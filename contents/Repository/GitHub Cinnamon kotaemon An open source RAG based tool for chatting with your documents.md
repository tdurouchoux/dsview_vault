## Summary

Cinnamon/kotaemon is an open-source RAG (Retrieval-Augmented Generation) tool designed for document interaction through chat interfaces. Key features include:

- **User Interface**: A minimalistic and clean design tailored for document question-answering (QA).
- **Compatibility**: Supports various Large Language Models (LLMs) from providers like OpenAI and Azure, as well as local models.
- **Installation Ease**: Simple installation scripts for quick setup, with options for Docker.
- **Multi-user Functionality**: Enables collaboration through multi-user logins and shared document collections.
- **Hybrid Retrieval Mechanism**: Combines full-text and vector retrieval for enhanced document access.
- **Multi-modal Support**: Capable of handling complex documents with figures and tables.
- **Citations**: Provides advanced citation capabilities, enhancing reliability in responses.
- **Customization**: Extensive options for developers to build tailored RAG pipelines and UI modifications through Gradio.

To install, users can choose between Docker or a manual Python setup, with specific instructions provided for system requirements and environment configurations. Developers are encouraged to contribute and customize various aspects of the application.
