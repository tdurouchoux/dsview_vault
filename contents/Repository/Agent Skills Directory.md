## Summary

The Agent Skills Directory is a platform for AI agent skills, which are reusable capabilities that can be installed with a single command to enhance agents with procedural knowledge. The directory features a leaderboard of skills, with the top skills including "find-skills" by vercel-labs, "vercel-react-best-practices" by vercel-labs, and "web-design-guidelines" by vercel-labs. The skills cover a wide range of topics, including web development, design, marketing, and more. The directory also allows users to search for skills and view trending and hot skills. The skills can be installed using the command `npx skillsadd<owner/repo>`.
