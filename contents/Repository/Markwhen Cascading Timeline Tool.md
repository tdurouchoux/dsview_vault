## Summary

Markwhen is an open-source tool that allows users to create cascading timelines from markdown-like text. It supports various date formats, including American, European, and ISO8601 styles, and can incorporate images, links, and locations. The main components include a web app (editor), a VSCode extension, and a view container for rendering timelines and calendars. Users can customize views using the view client library.

Key features:
- Markdown-like text input for timeline creation.
- Support for multiple date formats and media.
- Interactive editor available at markwhen.com.
- Option to self-host views and run locally using Docker.

Documentation and additional resources are available for setup and usage. The project is licensed under MIT.
