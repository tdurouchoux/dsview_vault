## Summary

Latent Scope is a tool designed for visualizing and exploring datasets through latent spaces, enabling users to better understand their data by embedding, projecting, clustering, and labeling it. It serves as a microscope for data, allowing for similarity searches, cluster exploration, and individual data point analysis while maintaining the context of the entire dataset.

Key features include:
- **Data Processing Workflow**: The tool processes input data through embedding, projecting, clustering, and labeling, resulting in structured annotations for further analysis.
- **Web UI and Python Interface**: Users can interact with the tool via a web interface or a Python API, making it flexible for different workflows.
- **Command Line Interface (CLI)**: A suite of command line scripts is available for users who prefer CLI over the web UI, facilitating data ingestion, embedding, clustering, and visualization.
- **Flat File Storage**: All data is stored in flat files, making it easy to export and manage outputs.
- **Experimentation and Exploration**: The tool records all choices made during the process, allowing users to revisit and analyze different paths taken during their exploration.

To get started, users need to install the latent-scope module, set up their environment, and follow the documentation for data ingestion and processing. The tool supports various embedding models and allows for easy integration of new models.
