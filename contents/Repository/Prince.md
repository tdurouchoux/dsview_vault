## Summary

Prince is a Python library for multivariate exploratory data analysis, offering methods like PCA, CA, MCA, MFA, FAMD, GPA, and PGA. It follows a scikit-learn API and uses Altair for visualization. A flowchart guides method selection based on data type (categorical, numerical, groups, shapes, or manifolds). Installation is via `pip install prince`. The library ensures correctness by testing against scikit-learn, FactoMineR, and geomstats. Citation for scientific use is provided. Created in 2016 and revamped in 2022, Prince has over 2 million downloads. Licensed under MIT.
## Links

- [FactoMineR](http://factominer.free.fr/) : FactoMineR is an R package for multivariate exploratory data analysis, which inspired the creation of the Prince library. It provides methods like PCA, CA, and MCA, similar to those implemented in Prince.
