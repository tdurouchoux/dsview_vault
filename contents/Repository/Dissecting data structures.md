## Summary

Datanomy is a terminal-based tool designed for inspecting and understanding data files, currently supporting Parquet files. It offers an interactive view of the data's structure, metadata, and internal organization. Key features include:

- **General Structure**: Visualize the overall layout of the Parquet file.
- **Schema**: Inspect the schema of the data.
- **Data**: View the actual data contained within the file.
- **Metadata**: Examine the metadata associated with the data.
- **Stats**: Access statistical information about the data.

The tool can be installed via PyPI or from the source, with support for both installation and direct usage without installation. It is built with Textual and PyArrow. Contributions are welcome, and the project is licensed under the Apache License 2.0.
