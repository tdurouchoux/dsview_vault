## Summary

Gitingest is a tool that allows users to convert any GitHub repository into a prompt-friendly text digest by simply replacing 'hub' with 'ingest' in the URL. Key features include:

- **Easy Code Context**: Generates a text digest from a Git repository URL or local directory.
- **Smart Formatting**: Outputs are optimized for large language model (LLM) prompts.
- **Statistics**: Provides insights on file structure, extract size, and token count.
- **CLI Tool**: Available for command-line usage (Linux only) and as a Python package.

Installation can be done via `pip install gitingest`, and it supports both command line and Python package usage. Users can also self-host the application using Docker.

Contributions are encouraged, and the project is designed to be accessible for first-time contributors. The repository is licensed under MIT.
