## Summary

thepipe is a Python package designed to extract clean data from various document types using vision-language models (VLMs). It provides functionalities to scrape markdown, tables, images, and structured data from complex sources such as PDFs, Word documents, videos, and more.

Key features include:
- Support for a wide range of file types (e.g., PDFs, Word, PowerPoint, videos, and HTML content).
- Built-in AI capabilities for document parsing and extraction.
- Flexible chunking methods to manage long documents for large language models (LLMs) handling.
- Simple installation and setup process, primarily relying on the OpenAI API by default.

The scraping functionality allows users to run commands to extract and format data, with options for customizing the extraction process. It integrates easily with LLMs for further processing and supports structured data extraction through specified schemas.

The tool is open-source and encourages contributions for enhancements.
