## Summary

dayplot is a Python library for creating calendar heatmaps, built on matplotlib for high customization. Key features include:

- **Installation**: Install via `pip install dayplot`.
- **Basic Usage**: Users can create heatmaps by importing the library and using the `dp.calendar()` function with parameters for dates, values, and customization options like colormaps and styles.
- **Examples**: The library provides various examples, including handling negative values, applying different colormaps, and customizing aesthetics such as box styles and colors.
- **Advanced Usage**: dayplot integrates seamlessly with matplotlib, allowing for extensive customization and integration with other plots.
- **Related Projects**: Other similar libraries include calplot, july, and calendarplot.

The documentation includes a gallery for additional examples and advanced usage scenarios.
