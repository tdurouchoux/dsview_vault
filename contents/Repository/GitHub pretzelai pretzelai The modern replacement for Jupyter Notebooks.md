## Summary

Pretzel is an open-source alternative to Jupyter Notebooks designed to enhance user experience through features like AI code generation, inline tab completion, and an AI sidebar for interactive assistance. It allows for seamless transitions from Jupyter with compatibility for existing configurations.

Key features include:
- **Installation**: Easily installable via pip or a Docker container, making it accessible on various platforms.
- **AI Integration**: Supports AI features for generating and editing code, error fixation, and coding assistance through prompts.
- **User Interaction**: Inline tab completion and an AI sidebar for asking questions and obtaining code snippets.
- **Configuration Options**: Supports switching between different AI models to optimize performance.
- **Roadmap**: Future plans include enhanced collaboration, SQL support, a visual analysis builder, and a more integrated coding experience akin to VSCode.

Pretzel prioritizes user privacy, not storing user code or data, and operates under the AGPLv3 license, promoting community contribution while protecting against commercial exploitation of the tool.
