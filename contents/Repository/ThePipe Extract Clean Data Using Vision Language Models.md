## Summary

thepipe is a Python package designed to extract clean data from various document types using vision-language models (VLMs). It can scrape markdown, tables, and images from complex documents and works seamlessly with large language models (LLMs) and vector databases. Key features include:

- Supports a wide range of input sources: PDFs, URLs, Word documents, PowerPoints, videos, audio, Jupyter notebooks, and more.
- Offers AI-native filetype detection and layout analysis for structured data extraction.
- Can be used via a hosted API or installed locally, requiring specific dependencies for full functionality.
- Provides functions for scraping and extracting data, allowing users to define schemas for structured data extraction.

The package is designed for ease of use, enabling quick setup and integration with existing AI workflows. It also includes support for multimodal inputs and can process content for downstream use with language models or vector databases.
