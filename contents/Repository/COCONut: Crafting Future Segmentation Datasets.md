## Summary

The GitHub repository bytedance/coconut_cvpr2024 presents COCONut, a large-scale human-verified dataset for segmentation, aimed at enhancing segmentation datasets with high-quality annotations. Key highlights include:

- **Dataset Availability**: COCONut is accessible on Kaggle and Hugging Face, with various splits (COCONut-S, COCONut-B, COCONut-L, relabeled COCO-val, and COCONut-val) offering a total of over 1.5 million masks across different image counts.
- **Tutorials**: Several tutorials are provided, covering topics such as semantic segmentation, instance segmentation, panoptic segmentation, open-vocabulary segmentation, and object detection.
- **Updates**: Regular updates include new annotations and tutorials, with recent releases for COCONut-val and instance segmentation annotations.
- **Usage**: The dataset is intended for research purposes only, following the same licensing as the COCO dataset, prohibiting commercial use.
- **Acknowledgments**: The project acknowledges contributions from various datasets and tools, including SAMA-COCO and detectron2.

For those interested in utilizing the dataset, instructions for downloading annotations and images are provided, along with a FAQ section addressing common issues.
