## Summary

Cinnamon/kotaemon is an open-source tool designed for document interaction using Retrieval-Augmented Generation (RAG). It features a user-friendly interface for both end users and developers, allowing for document question-answering (QA) and the creation of custom RAG pipelines.

Key features include:
- Support for various LLM API providers and local models.
- A hybrid RAG pipeline that combines full-text and vector retrieval.
- Multi-modal QA capabilities, including support for documents with figures and tables.
- Advanced citation features that provide detailed references for LLM responses.
- Configurable settings for retrieval and generation processes.
- Extensibility through Gradio, allowing for UI customization.

Installation options include Docker and manual setup, with detailed instructions provided for both methods. The tool supports various document types and offers a framework for building custom reasoning and indexing pipelines. The project is actively developed, welcoming feedback and contributions.
