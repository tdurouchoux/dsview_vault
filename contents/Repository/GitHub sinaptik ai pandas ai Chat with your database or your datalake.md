## Summary

PandasAI is a Python library designed to facilitate conversational data analysis, enabling users to interact with databases or data lakes (SQL, CSV, Parquet) using natural language. It leverages large language models (LLMs) and Retrieval-Augmented Generation (RAG) for data queries.

Key features include:
- **Ease of Use**: Non-technical users can easily query data; technical users can save time and effort.
- **Data Interaction**: Users can load data as DataFrames and ask questions using natural language.
- **Installation**: The library can be installed with pip or poetry.
- **Multiple DataFrames**: Supports querying multiple DataFrames simultaneously.
- **Visualization**: Capable of generating charts from requests.
- **Docker Sandbox**: Offers a secure environment for running queries to mitigate risks.

Python requirements include version 3.8+, and the library is available under the MIT license. The project is currently in beta, with upcoming features and support open for community contributions.
