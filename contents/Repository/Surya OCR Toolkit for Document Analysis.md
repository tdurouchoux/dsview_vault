## Summary

Surya is an open-source OCR toolkit designed for document processing, capable of handling over 90 languages. Key features include:

- **OCR Capabilities**: Performs line-level text detection, layout analysis, reading order detection, and table recognition.
- **Performance**: Benchmarks show Surya performs favorably against cloud services like Google Cloud Vision and Tesseract, with a focus on speed and accuracy.
- **Installation**: Requires Python 3.10+ and PyTorch. Installation can be done via pip, and model weights are automatically downloaded.
- **Usage**: Supports various document formats (PDFs, images, etc.) and provides command-line tools for OCR, layout detection, reading order, and table recognition.
- **Community and Support**: Active community discussions on Discord and a focus on making the toolkit accessible for both research and commercial use, with specific licensing terms for organizations.
- **Performance Tips**: Users can optimize performance by adjusting batch sizes and other environment variables based on their hardware capabilities.

Surya is particularly suited for printed text and document OCR, with limitations on handwriting and non-document images. It includes troubleshooting tips for improving OCR results and offers a manual installation option for developers.
