## Summary

Pydoll is an innovative library for automating Chromium-based browsers without relying on a WebDriver, enhancing realism in interactions. Key features include:

- **WebDriver Elimination**: It bypasses the complexity associated with WebDriver compatibility.
- **Async Performance**: Enables lightning-fast automation with asynchronous capabilities.
- **Human-Like Interactions**: Mimics real user behavior.
- **Multi-Browser Support**: Compatible with Chrome and Edge.

Upcoming updates will refactor module structures, introduce static typing, improve logging, and enhance HTML interaction capabilities. Key functionalities such as proxy management and advanced event handling are also included. The installation is straightforward via pip, and comprehensive documentation is provided for usage and troubleshooting.
