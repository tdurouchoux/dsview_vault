## Summary

Devstral-Small-2505 is an agentic large language model (LLM) designed for software engineering tasks, developed through collaboration between Mistral AI and All Hands AI. Key features include:

- **Performance**: Ranked #1 in the SWE-bench benchmark with a score of 46.8%, outperforming larger models like GPT-4 and Claude.
- **Parameters**: Contains 24 billion parameters, allowing for efficient local deployment on devices like RTX 4090.
- **Context Window**: Supports a substantial context window of 128k tokens.
- **APIs and Tools**: Recommended usage with OpenHands, vLLM, and several deployment options like Docker.
- **Licensing**: Distributed under the Apache 2.0 license for flexible use.

Devstral is user-friendly for coding tasks; users can prompt it to generate applications (e.g., a To-Do list app) or analyze code coverage in repositories. The model utilizes various libraries and tools, encouraging integration into production environments. Advanced interactions include setting up servers, integrating with the OpenHands platform, and employing multiple inference methods.
