## Summary

Quivr is an opinionated Retrieval-Augmented Generation (RAG) framework designed for easy integration of Generative AI into applications. It supports various Large Language Models (LLMs) such as GPT-4, Groq, and Llama, and can work with different vector stores like PGVector and Faiss. Key features include:

- Customizable RAG that allows users to focus on product development.
- Compatibility with multiple file types (PDF, TXT, Markdown) and the ability to add custom parsers.
- Integration with Megaparse for file ingestion.
- Simple installation and setup process, requiring only Python 3.10 or newer.

The framework allows users to create RAG workflows with minimal code and provides configuration options for API keys, workflow definitions, and LLM settings. Users can enhance their applications by adding internet search capabilities and other tools. The documentation offers guidance for getting started and advanced configurations.
