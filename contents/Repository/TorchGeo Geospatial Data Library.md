## Summary

TorchGeo is a PyTorch library designed for geospatial data, offering datasets, samplers, transforms, and pre-trained models. It aims to simplify the integration of machine learning with geospatial data for both machine learning and remote sensing experts.

Key features include:

- **Installation**: Recommended via pip (`pip install torchgeo`), with additional instructions for conda and spack.
- **Datasets and Samplers**: Supports various geospatial datasets, allowing users to combine and sample data from different sources while handling coordinate reference systems automatically.
- **Benchmark Datasets**: Includes datasets for tasks like image classification, semantic segmentation, and object detection, compatible with PyTorch data loaders.
- **Pre-trained Weights**: Offers models pre-trained on multispectral sensors, facilitating transfer learning for remote sensing applications.
- **Reproducibility with Lightning**: Provides Lightning datamodules for structured training and evaluation, simplifying the process of running experiments.
- **Command-Line Interface**: Supports training and validation through a command-line interface, allowing configuration via YAML/JSON files.

The library encourages contributions and provides a citation for academic use. Documentation is available on ReadTheDocs, including API details and tutorials.
