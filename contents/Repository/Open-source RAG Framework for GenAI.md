## Summary

Quivr is an open-source framework designed for building Generative AI (GenAI) productivity assistants, enabling users to interact with various document types (PDF, CSV, etc.) and applications. Key features include:

- **Fast and Efficient**: Optimized for quick data access.
- **Secure**: Users maintain control over their data.
- **Cross-Platform Compatibility**: Works on Ubuntu 20 or newer.
- **File Compatibility**: Supports multiple formats including text, Markdown, and multimedia.
- **Open Source**: Free to use and modify.
- **Public/Private Sharing**: Options to share or keep data private.
- **Offline Mode**: Access data without an internet connection.

The framework utilizes technologies like Langchain and various LLMs (e.g., GPT-3.5/4, Anthropic) for enhanced functionality. Installation requires Docker and Docker Compose, with a straightforward setup process. Users can deploy it locally or on cloud platforms. The project is actively maintained, with contributions welcomed from the community.
