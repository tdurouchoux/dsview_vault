## Summary

Maxun is an open-source, no-code platform designed for web data extraction, allowing users to convert websites into APIs and spreadsheets effortlessly. It features a user-friendly interface to create custom robots that automate data scraping tasks, such as capturing lists, text, or screenshots.

Key functionalities include:
- **No-Code Extraction**: Users can set up data extraction workflows without programming knowledge.
- **Custom Robots**: Robots can be trained in minutes to perform repetitive tasks.
- **Proxy Support**: Users can integrate personal proxies to navigate anti-bot protections.
- **Integration Capabilities**: Planned support for Google Sheets and Airtable integrations.

Deployment is straightforward, with instructions for using Docker or manual installation that includes setting up environment variables for configuration.

The project is under active development and licensed under AGPLv3, with community feedback welcomed for improvements. Current features are focused on streamlined data extraction processes and adaptable robot behavior to handle various web layouts and login scenarios.
