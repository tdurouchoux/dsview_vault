## Summary

Pretzel is an open-source alternative to Jupyter Notebooks, designed to enhance its capabilities with features like AI code generation, inline tab completion, and an AI sidebar for real-time assistance. It allows for easy migration from Jupyter, maintaining compatibility with existing configurations and extensions.

Key features include:
- **Installation**: Can be installed via pip or run in a Docker container.
- **AI Integration**: Offers AI-assisted coding through commands like "Ask AI" and inline completions triggered by typing.
- **Collaboration**: Plans for real-time collaboration features and SQL support.
- **Configuration**: Users can customize AI models and settings, with support for various AI services.
- **Privacy**: No personal data is collected; only anonymous telemetry for AI features is used.

The roadmap includes advanced features like a visual analysis builder and a VSCode-like coding experience. The project is licensed under AGPL-3.0 to ensure open-source contributions while preventing commercial exploitation.
