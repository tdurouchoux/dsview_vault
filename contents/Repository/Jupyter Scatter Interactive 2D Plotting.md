## Summary

Jupyter Scatter is an interactive 2D scatter plot widget designed for Jupyter Lab, Notebook, and Google Colab, capable of handling millions of data points. Key features include:

- **Interactivity**: Users can pan, zoom, and select data points using a mouse or Python API.
- **Scalability**: Utilizes WebGL rendering to smoothly plot millions of points.
- **View Linking**: Allows synchronization of view, hover, and selection across multiple scatter plot instances.
- **Effective Defaults**: Automatically selects perceptually effective colors and opacities for data points.
- **User-Friendly API**: Integrates well with Pandas DataFrames and offers a readable API.

The widget supports various functionalities such as customizing visual encodings (color, size, opacity), linking multiple scatter plots, and visualizing large datasets. Installation can be done via pip, and it includes examples for quick start, advanced usage, and linking plots. For development, it requires specific versions of uv and Node.js, and it supports testing and auto-reloading features.

For detailed documentation and examples, users can visit the official site at jupyter-scatter.dev.
