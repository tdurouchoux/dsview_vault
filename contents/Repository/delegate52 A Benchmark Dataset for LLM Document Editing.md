## Summary

DELEGATE52 is a benchmark dataset from Microsoft on Hugging Face, designed to evaluate LLMs on long-horizon delegated document editing across 52 professional domains (e.g., accounting, music notation, coding). It contains 234 work environments (48 domains) with real-world seed documents and edit tasks, structured as round-trip relay simulations. Key features: 1,629 edit pairs, distractor context for robustness testing, and metadata like token counts and licenses. The dataset is released under CDLA Permissive 2.0 and aims to study AI delegation workflows. It includes code snippets for loading and running evaluations, and contributors can expand it via GitHub. Limitations: English-only, no PII, and not for high-risk applications. Paper: [arxiv:2604.15597](https://arxiv.org/abs/2604.15597).
