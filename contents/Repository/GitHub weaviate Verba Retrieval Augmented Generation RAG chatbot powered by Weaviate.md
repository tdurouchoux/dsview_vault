## Summary

Verba is an open-source Retrieval Augmented Generation (RAG) chatbot powered by Weaviate, designed for seamless data querying and interaction. It provides an end-to-end interface for exploring datasets and extracting insights, supporting multiple local and cloud deployment options with models from various LLM providers like OpenAI, Anthropic, and Cohere.

Key features include:
- Support for diverse data sources: PDFs, CSVs, GitHub, etc.
- Semantic and keyword search capabilities, customizable metadata, and advanced querying.
- Chunking techniques to handle different file formats.
- Integration with various embedding and generation models, allowing flexibility based on user needs.
- Docker support for deployment and a customizable frontend interface.

To get started, users can install Verba via pip, build it from source, or deploy using Docker, with configuration requirements for API keys depending on the services utilized. Community contributions are encouraged, and the project welcomes feedback on performance and usability.
