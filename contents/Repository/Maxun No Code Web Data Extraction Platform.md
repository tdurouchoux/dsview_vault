## Summary

Maxun is an open-source, no-code web data extraction platform that allows users to convert websites into APIs and spreadsheets using automated robots. Key features include:

- **No-Code Data Extraction**: Users can create robots to scrape data without coding.
- **Robot Actions**: Robots can perform actions like capturing lists, text, and screenshots.
- **Proxy Support**: Users can connect external proxies to bypass anti-bot measures.
- **Scheduling**: Robots can be scheduled to run at specific times.
- **Adaptability**: Future updates will allow robots to adapt to website layout changes and extract data behind logins with two-factor authentication.
- **Cloud Version**: A managed cloud service is available for large-scale data extraction, handling anti-bot detection and proxy management.

Installation can be done via Docker or manually, requiring environment variables for configuration. The project is in early development, and user feedback is encouraged. It is licensed under AGPL-3.0.
