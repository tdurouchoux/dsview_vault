## Summary

Local Deep Research is an AI-powered assistant designed to convert complex questions into detailed, cited reports through iterative analysis using various large language models (LLMs). It integrates diverse knowledge sources, including academic databases, scientific repositories, and private documents.

Key features include:
- **Advanced Research Capabilities**: Automated deep research with intelligent follow-up questions and comprehensive coverage.
- **Flexible LLM Support**: Compatibility with local models (via Ollama) and cloud-based models (e.g., Claude, GPT).
- **Rich Output Options**: Detailed findings, structured reports, and quick summaries with proper citations.
- **Privacy-Focused**: Operates locally for enhanced privacy, with configurable search settings.
- **Enhanced Search Integration**: Intelligent selection of search engines and local document search capabilities.
- **Academic & Scientific Integration**: Direct access to sources like PubMed and arXiv, with formatted citations.

The system supports various installation methods, including Docker and local setups, and offers a command-line interface as well as a web interface for user interaction. It also features a unified database for configuration management and supports multiple LLM providers and search engines. 

Contributors are encouraged to participate in the development process, and the project is licensed under the MIT License.
