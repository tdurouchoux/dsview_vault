## Summary

Jupyter Scatter is an interactive 2D scatter plot widget designed for Jupyter Lab, Notebook, and Google Colab, capable of handling millions of data points. Key features include:

- **Interactivity**: Users can pan, zoom, and select points via mouse or Python API.
- **Scalability**: Utilizes WebGL rendering for smooth visualization of large datasets.
- **View Linking**: Allows synchronization of view, hover, and selection across multiple plots.
- **Effective Defaults**: Automatic selection of perceptually effective colors and opacities.
- **API Integration**: A user-friendly API that seamlessly interacts with Pandas DataFrames.

The widget supports interactive dataset exploration, enabling dynamic adjustments to visual encodings like size and color based on data characteristics. It offers a function-based API for grouping customization options and facilitates linking of multiple scatter plots for comparative analysis. Installation can be done via pip, and it is also compatible with Google Colab.

For development, setup requirements include a specific version of `uv` and Node.js. Users are encouraged to cite the relevant JOSS paper for academic use.
