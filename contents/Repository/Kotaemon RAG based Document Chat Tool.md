## Summary

Cinnamon's Kotaemon is an open-source tool designed for document interaction using Retrieval-Augmented Generation (RAG) techniques. It features a user-friendly interface for end users and a customizable framework for developers to build their own RAG pipelines. Key functionalities include:

- **User Interface**: Clean and minimalistic design for easy document QA.
- **LLM Compatibility**: Supports various large language models (LLMs) from providers like OpenAI and Azure, as well as local models.
- **Installation Options**: Available via Docker or direct installation with Python.
- **Document Management**: Allows multi-user access, file organization, and collaboration.
- **Hybrid RAG Pipeline**: Combines full-text and vector retrieval for improved accuracy.
- **Multi-modal Support**: Handles various document types, including those with figures and tables.
- **Advanced Reasoning**: Supports complex question answering and reasoning methods.
- **Customizability**: Built on Gradio, enabling extensive UI customization and integration of different indexing and retrieval strategies.

The tool is actively developed, with a focus on community contributions and feedback. Installation requires Python 3.10 or higher, and users can choose between a lightweight or full Docker image for deployment.
