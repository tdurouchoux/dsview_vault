## Summary

Validoopsie is a lightweight and user-friendly data validation library for Python, designed to simplify the process of declaring classes and chaining validations, similar to popular DataFrame libraries. It supports the "Bring Your Own DataFrame" (BYOD) concept, allowing users to validate data using any compatible DataFrame.

Key features include:
- Easy installation via pip.
- Intuitive syntax that encapsulates each validation function in its own method, promoting readability and simplicity.
- Inspired by the Great Expectations library, it aims to streamline data validation tasks.
- Users can chain validations together using the Validate class, making it straightforward to implement checks like column equality, uniqueness, and value ranges.
- Validation results are presented in a structured JSON format, detailing the status and specifics of each validation.

The library is licensed under MIT and encourages contributions and custom validation development.
