## Summary

Project Glasswing is a collaborative initiative involving AWS, Anthropic, Apple, Microsoft, NVIDIA, and others to secure critical software using Anthropic’s frontier AI model, **Claude Mythos Preview**. This model has demonstrated advanced coding and cybersecurity capabilities, autonomously identifying thousands of high-severity vulnerabilities—including decades-old flaws in major OS/browsers—that evaded prior detection methods.

Key technical points:
- **AI-driven cyber threats**: Mythos Preview outperforms prior models (e.g., 83.1% vs. 66.6% on CyberGym benchmarks) in finding/exploiting vulnerabilities, reducing the barrier to cyberattacks.
- **Defensive focus**: Partners (e.g., Cisco, AWS, Microsoft) will use the model to proactively scan and patch critical systems, with Anthropic committing $100M in usage credits and $4M in donations to open-source security orgs.
- **Urgency**: The model’s speed (e.g., chaining Linux kernel vulnerabilities) highlights the need for rapid industry-wide adoption of AI-augmented security.
- **Long-term goals**: Plans include vulnerability disclosure frameworks, secure-by-design practices, and collaboration with governments to mitigate risks. A third-party body may oversee future efforts.

**Takeaway**: AI is both a threat multiplier and a defense enabler; Project Glasswing aims to harness it to stay ahead of adversaries.
