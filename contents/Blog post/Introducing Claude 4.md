## Summary

Claude 4 introduces two new models from Anthropic: Claude Opus 4 and Claude Sonnet 4, both enhancing coding, advanced reasoning, and AI agent capabilities. Claude Opus 4 is noted as the top coding model, showing exceptional performance in complex tasks and agent workflows, and is able to utilize tools and maintain continuity in long-running operations. Claude Sonnet 4 provides improved coding capabilities and better instruction-following than its predecessor, Sonnet 3.7.

Key features include:
- **Extended Thinking with Tool Use**: Both models can alternate between reasoning and tool use, improving response quality.
- **Model Capabilities**: They support parallel tool usage and enhanced memory functionalities when local files are accessible, allowing for sustained task awareness.
- **Claude Code Availability**: This SDK enables developers to integrate Claude's capabilities into their development environments, streamlining workflows in IDEs like VS Code and JetBrains.
- **Hybrid Models**: They offer modes for near-instant responses and in-depth reasoning.

Performance highlights:
- Claude Opus 4 leads on key benchmarks like SWE-bench, achieving a score of 72.5%, while Sonnet 4 follows closely with 72.7%.
- Enhanced model behavior reflects fewer shortcuts in tasks and improved memory retention for task continuity.

The models are integrated into various platforms, including Anthropic API, Amazon Bedrock, and Google Cloud's Vertex AI, with consistent pricing similar to previous versions.
