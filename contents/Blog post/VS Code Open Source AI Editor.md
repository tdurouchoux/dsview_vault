## Summary

Visual Studio Code (VS Code) is transitioning to being an open-source AI editor. The team plans to open source the code for the GitHub Copilot Chat extension under the MIT license, integrating relevant components into the core of VS Code. This shift aligns with their philosophy of being open, collaborative, and community-driven, responding to advancements in AI, community feedback, and the growing ecosystem of open-source tools.

Key motivations for this transition include:
1. Improvements in large language models reducing the need for proprietary strategies.
2. Common UX patterns emerging across editors, which the community can refine.
3. The desire for transparency regarding data collected by AI tools.
4. The importance of community involvement in addressing security issues promptly.

Next steps involve open-sourcing the Copilot Chat code, refactoring AI features into the core of VS Code, and making the prompt test infrastructure available for community contributions. The goal is to facilitate ease of contribution while maintaining robust performance and a user-friendly interface. Updates will be shared through their iteration plan and FAQs.
