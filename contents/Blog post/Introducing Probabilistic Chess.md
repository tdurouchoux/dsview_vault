## Summary

Probabilistic Chess is a new variant that introduces success probabilities to chess moves, adding an element of unpredictability and strategy. Key features include:

- **Movement Probability**: Each square has a defined success probability for moves, determined by a dice roll.
- **Winning Condition**: Capturing the opponent's king is the goal, not just checkmating.
- **Dynamic Gameplay**: Players can make moves while in check, and stalemates are eliminated, ensuring every game ends in a win.
- **Variants**: Different gameplay variants are being tested, such as modifying king move success rates to balance gameplay.
- **Implementation**: The game is built using HTML, CSS, and JavaScript, which presents challenges for a chess AI that needs to account for probabilities.
- **AI Development**: The AI incorporates traditional chess algorithms and adapts them to probabilistic mechanics. There are modes for varying difficulty using algorithms like Alpha-Beta with Expectation-Maximization.
- **Ongoing Development**: The game is in an experimental alpha phase, inviting user feedback for improvements.

The blend of chance and skill transforms traditional chess dynamics, making each game unique.
