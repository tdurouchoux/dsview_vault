## Summary

Anthropic has announced the upgraded Claude 3.5 Sonnet and the new Claude 3.5 Haiku models. Claude 3.5 Sonnet shows significant improvements in coding performance, particularly in agentic coding and tool use tasks, outperforming all publicly available models. It has enhanced capabilities for software engineering, with notable feedback from companies like GitLab and Cognition.

Claude 3.5 Haiku, set to release later this month, offers high speed and improved performance across various tasks, especially coding, making it suitable for user-facing applications. 

A key new feature is the public beta of "computer use," allowing Claude to interact with computer interfaces like a human. This capability is still experimental and may be error-prone, but it enables automation of complex tasks. Developers can utilize this feature through the Anthropic API, Amazon Bedrock, and Google Cloud’s Vertex AI.

Safety measures are in place to mitigate risks associated with this new functionality, and feedback from early users will guide future improvements.
