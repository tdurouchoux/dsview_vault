## Summary

Anthropic has announced the upgraded Claude 3.5 Sonnet and the new Claude 3.5 Haiku models. Claude 3.5 Sonnet shows significant improvements in coding performance, particularly in agentic coding tasks, with scores surpassing all publicly available models. It is now available for all users and includes a new capability called "computer use," allowing it to interact with computer interfaces like a human. This feature is in public beta and is still experimental, with early adopters like Replit and Canva exploring its potential.

Claude 3.5 Haiku, set to release later this month, offers enhanced performance at a similar cost and speed to its predecessor, excelling in coding tasks and suitable for user-facing applications. Both models are available through the Anthropic API, Amazon Bedrock, and Google Cloud’s Vertex AI.

The computer use capability allows developers to automate tasks by translating instructions into computer commands, although it currently faces challenges with certain actions. Anthropic emphasizes a proactive approach to safety, implementing classifiers to monitor potential misuse. The company encourages feedback from developers to refine these technologies as they evolve.
