## Summary

Anthropic has announced two new AI models: Claude 3.5 Sonnet and Claude 3.5 Haiku, along with a public beta feature called computer use. Claude 3.5 Sonnet shows significant improvements in coding tasks, particularly in agentic coding, scoring 49.0% on SWE-bench Verified, outperforming all available models in this area. Claude 3.5 Haiku, launching later this month, is designed for speed and affordability, excelling in coding as well, with a score of 40.6% on the same benchmark.

The computer use feature allows Claude to interact with computer interfaces similarly to humans, enabling automation of complex tasks. Although still experimental, it demonstrates a promise for a range of applications including software development and research. Initial testing has shown Claude 3.5 Sonnet performing notably better than competitors in computer tasks. 

Feedback from users like GitLab and Cognition indicates meaningful advancements in various use cases compared to previous versions. However, Anthropic is taking a cautious approach in deploying this technology to mitigate potential risks such as misinformation or fraud. Overall, these developments aim to enhance user interaction and automation capabilities.
