## Summary

The Gemini API has introduced a new experimental text embedding model (gemini-embedding-exp-03-07) that outperforms the previous state-of-the-art model and ranks first on the Massive Text Embedding Benchmark (MTEB) Multilingual leaderboard. This model is designed for a variety of applications, including intelligent retrieval, recommendation systems, text classification, and more, by capturing semantic meaning through numerical representations.

Key features include:
- An input token limit of 8,000 tokens, allowing for larger text embeddings.
- Output dimensions of 3,000, providing high-dimensional embeddings.
- Matryoshka Representation Learning (MRL) for scalable storage.
- Support for over 100 languages, doubling previous capabilities.
- A unified model that surpasses earlier task-specific models.

The model is currently in an experimental phase, with plans for a stable release in the future. Developers can access it via the Gemini API, and feedback is encouraged.
