## Summary

Probabilistic Chess is a new variant of chess that introduces success probabilities for each move, making the game more strategic and unpredictable. Key features include:

- **Movement Probability**: Each square has a defined probability of successful movement, determined by a dice roll.
- **Winning Condition**: The game is won by capturing the opponent's king, not by checkmate.
- **Dynamic Gameplay**: Players can make moves while in check and can even put their own king in check, adding complexity to strategy.
- **Guaranteed Wins**: All games end in a win, eliminating draws and stalemates, which enhances the excitement of the game.
- **Variants**: Different gameplay variants are being tested, such as ensuring king moves always succeed or have a higher success rate.
- **Implementation**: The game is built using HTML, CSS, and JavaScript, with a focus on performance and low server costs. AI opponents are developed using Stockfish and custom algorithms to account for probabilities.
- **Ongoing Development**: The game is in an experimental phase, with plans for improvements based on user feedback.

Overall, Probabilistic Chess aims to provide a fresh and engaging twist on traditional chess, emphasizing probabilistic thinking and strategic depth.
