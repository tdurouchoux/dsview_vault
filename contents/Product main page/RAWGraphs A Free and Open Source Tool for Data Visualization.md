## Summary

RAWGraphs is an open-source, web-based data visualization tool designed for data scientists and designers. It is built on top of open-source projects and encourages community contributions. Key features include:

- **Wide Range of Charts**: Offers nearly 30 visual models to represent quantities, hierarchies, time series, and other data insights.
- **Data Privacy**: Processes data locally within the user's web browser, ensuring data security.
- **Export Options**: Allows users to save projects or export visualizations as vector or raster images for further editing in preferred software.

The tool is supported by sponsors and is maintained by DensityDesign, Calibro, and Inmagik. Additional resources include tutorials, courses, and a gallery of custom charts.
