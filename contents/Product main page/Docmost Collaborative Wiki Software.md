## Summary

Docmost is an open-source collaborative wiki and documentation software designed for seamless knowledge management. Key features include:

- **Real-time Collaborative Editor**: Users can edit pages simultaneously with support for rich text, diagrams, tables, and LaTeX.
- **Diagram Support**: Built-in editors for Mermaid, Draw.io, and Excalidraw.
- **Permissions System**: Robust controls for viewing, editing, and managing content to ensure security.
- **Spaces**: Organize content into dedicated areas for different teams or projects.
- **Groups**: Manage user groups for unified permission settings.
- **Comment System**: Inline commenting for discussions on pages.
- **Page History**: Track changes and revert to previous versions.
- **Search Functionality**: Powered by Postgres full-text search for quick information retrieval.
- **Nested Pages**: Support for organizing pages hierarchically with drag-and-drop management.
- **Attachments**: Easy file and image attachment with support for S3 and local storage.

Docmost facilitates effective collaboration and documentation management across teams.
