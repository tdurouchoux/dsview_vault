## Summary

GeoUtil is a browser-based, free toolkit for working with geographic data, offering a range of functionalities without requiring server uploads or sign-ups. Key features include:

1. **Measurement Tools**:
   - Distance Tool: Calculates great-circle distances between points.
   - Globe Distance Tool: Visualizes distances on a 3D globe.
   - Area Tool: Measures surface areas of regions.
   - Bearing/Azimuth Tool: Calculates compass bearings between points.
   - Feature Counter/Stats: Analyzes GeoJSON statistics.

2. **Format Converters**:
   - Converts between GeoJSON, TopoJSON, WKT, Shapefile, KML, and CSV.
   - GeoJSON to SVG/PNG/JPEG for image exports.

3. **Coordinate Tools**:
   - CRS Converter: Transforms coordinates between over 500 CRS.
   - Coordinate Format Converter: Converts between DD, DMS, DDM, UTM, and MGRS.
   - Coordinate Extractor: Extracts coordinates from GeoJSON files.

4. **File Mergers**:
   - Merges GeoJSON, TopoJSON, JSON, and Shapefile files.

5. **Data Splitters**:
   - Splits GeoJSON and TopoJSON files by features or properties.

6. **Data Minifiers**:
   - Minifies GeoJSON, TopoJSON, and JSON files to reduce size.

**Privacy & Security**:
- All tools run locally in the browser, ensuring data never leaves the device.
- Works offline after initial load and provides complete privacy.

**Use Cases**:
- GIS professionals, web developers, data analysts, educators, and researchers.
- Converting formats, merging datasets, transforming coordinates, validating data, and more.

**Getting Started**:
- Choose a tool, upload or paste data, configure options if needed, and download results.
- Most tools offer live previews for verification before downloading.

**Support**:
- Detailed documentation, FAQs, and a Reddit community for questions and feedback.
