## Summary

Gephi Lite is a web-based, open-source tool for network visualization and analysis. Key features include: no installation required (works in modern browsers), local data processing (no server), GitHub Gist integration for saving/loading graphs, and customizable graph appearance. It supports dynamic node/edge styling, data table exploration, and embedding via iFrame or JavaScript driver. Mobile-friendly with touch/multitouch support and responsive UI. Licensed under GPL-3.0, with documentation and tutorials available. Citation: Girard et al. (2025).
