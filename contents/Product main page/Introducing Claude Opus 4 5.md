## Summary

Claude Opus 4.5, the latest model from Anthropic, is now available and excels in coding, agents, and computer use, with improved performance in everyday tasks like deep research and working with slides and spreadsheets. It is state-of-the-art in real-world software engineering tests and is available on major cloud platforms with updated pricing. The model is praised for handling ambiguity, reasoning about tradeoffs, and fixing complex bugs. It outperforms previous models in coding tasks, token efficiency, and long-horizon autonomous tasks. Opus 4.5 also shows improvements in safety, being more robust against prompt injection attacks. The Claude Developer Platform has been updated with new tools for longer-running agents and better integration with Excel, Chrome, and desktop applications. The model's capabilities include better vision, reasoning, and mathematics skills, and it is state-of-the-art in many domains. The release includes updates to Claude Code, with Plan Mode and desktop app availability, and improvements in handling long conversations. Usage limits have been updated to accommodate Opus 4.5's capabilities.
