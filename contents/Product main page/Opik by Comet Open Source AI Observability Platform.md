## Summary

Opik by Comet is an open-source AI observability platform designed for agentic AI systems. Key features include:

- **End-to-end observability**: Logs and visualizes every step of an agent's workflow (user interactions, tool calls, etc.) with automated evaluation workflows to identify and fix errors across development, testing, and production.
- **LLM-as-a-Judge metrics**: Evaluates agent performance using 30+ metrics (e.g., answer relevance, hallucination) with plain-text assertions or reference datasets.
- **Production monitoring**: Real-time evaluation of production traces, alerting for failures, and tracking token usage/cost optimization.
- **Automated debugging**: Uses plain-text assertions in Test Suites and Opik’s coding assistant (Ollie) to auto-implement fixes and regression tests.
- **Agent Playground**: Tests agents end-to-end with configurable models, prompts, and parameters.
- **Prompt Optimizer**: Offers 7 algorithms for prompt optimization.
- **Open-source core**: Free core features with scalable enterprise options; integrates with GitHub.

Opik supports the full agent development lifecycle, from prototype to production, with collaboration tools for governance and debugging. Free tier available.
