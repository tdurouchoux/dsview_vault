## Summary

Langfuse is an open-source LLM engineering platform for tracing, prompt management, evaluations, and analytics. Key features include hierarchical tracing, LLM-as-a-judge evaluations, prompt versioning, and human-in-the-loop workflows. It integrates with 80+ frameworks (e.g., LangChain, LlamaIndex) and model providers (OpenAI, Anthropic), using OpenTelemetry and ClickHouse for scalable observability. The platform supports self-hosting (Docker, Kubernetes, cloud) and offers a free tier (50k observations/month). Notable users include Canva and 19 Fortune 50 companies, with 10+ billion observations processed monthly. It emphasizes open standards, async processing, and agent-friendly tooling (CLI, MCP servers).
