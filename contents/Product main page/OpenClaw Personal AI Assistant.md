## Summary

OpenClaw is an open-source, hackable personal AI assistant that runs on your machine (macOS, Windows, Linux) with persistent memory and proactive capabilities. It integrates with any chat app (WhatsApp, Telegram, Discord, etc.) and can autonomously perform tasks like managing emails, calendars, flights, running tests, generating code, handling APIs, and even controlling hardware (e.g., air purifiers via voice commands).

Key technical points:
- Runs locally (private by default) with support for Anthropic, OpenAI, or local models.
- Extendable via community-built skills or plugins, including self-generated ones.
- Full system access (file operations, shell commands) with optional sandboxing.
- Persistent memory and persona customization.
- Browser control and automation (web scraping, form filling).
- Companion app (macOS) for menubar access.

Users highlight its transformative potential, comparing it to early AGI experiences, with capabilities like autonomous coding loops, multi-agent coordination, and real-time task automation. It is praised for being open-source, self-hackable, and hostable on-prem, contrasting with conventional SaaS.

Partners with VirusTotal for secure skill integrations. Community actively builds and shares new skills via ClawHub.

Quick start:
- One-liner npm installer (works everywhere, includes Node.js).
- `openclaw onboard` for setup.
- Hackable macOS installer also available via GitHub.

Works with 50+ integrations (e.g., Gmail, GitHub, Spotify, Hue, Obsidian, Twitter, Browser).
