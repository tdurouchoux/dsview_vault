## Summary

Cursor 3 introduces a unified workspace for AI-assisted software development, shifting from manual file editing to agent-driven workflows. Key updates include:

- **Multi-agent management**: All agents (local/cloud) consolidated in a sidebar, supporting parallel execution across repositories.
- **Seamless cloud/local transitions**: Agents can move between environments (e.g., cloud for offline tasks, local for testing with Composer 2).
- **Enhanced workflow**: New diff view simplifies commit-to-PR pipeline management.
- **Core features**: Native multi-repo support, integrated browser for web-based tasks, and plugin marketplace (MCP/skills/sub-agents).
- **IDE integration**: Maintains Cursor’s LSP support and file navigation while prioritizing agent collaboration.

Focus: Scaling from individual agent oversight to autonomous agent fleets, with Cursor 3 as a bridge between current IDEs and future agentic development. Access via `Cmd+Shift+P -> Agents Window`.
