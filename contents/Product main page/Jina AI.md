## Summary

Jina AI offers a suite of tools for enhancing search and retrieval systems, including:

- **Reader**: Converts URLs to LLM-friendly Markdown (e.g., `r.jina.ai/https://example.com`) with customizable extraction (CSS selectors, JS execution, etc.).
- **Embeddings**: Multimodal/multilingual embeddings (e.g., `jina-embeddings-v5`) for retrieval tasks.
- **Reranker**: Optimizes search relevancy (e.g., `jina-reranker-v3`).

Key features:
- SOC 2 compliant, no credit card required for API access.
- High-rate limits with API keys.
- Supports dynamic content (Shadow DOM, iframes) and EU compliance.
- Publications in top-tier conferences (NeurIPS, ICML, etc.).

API endpoints:
- `r.jina.ai` (Reader), `s.jina.ai` (web search), `mcp.jina.ai` (MCP server for LLMs).
- Customizable parameters (timeout, locale, caching, etc.).

Enterprise-focused with offices in Sunnyvale (CA) and Berlin.
