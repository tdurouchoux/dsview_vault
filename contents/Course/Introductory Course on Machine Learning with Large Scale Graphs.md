## Summary

The notes serve as an introductory course on machine learning with large-scale graphs, paralleling the content of Stanford's CS224W course. Divided into three main areas—preliminaries, network methods, and machine learning with networks—the course covers a range of topics.

**Preliminaries** include understanding graph structures, measuring networks, and motifs. 

**Network Methods** focus on identifying structural roles, spectral clustering, influence maximization, outbreak detection, link analysis, network effects, robustness, evolution, and knowledge graphs.

**Machine Learning with Networks** addresses message passing, node classification, node representation learning, graph neural networks, and generative models for graphs.

The course encourages contributions to the notes via GitHub, offering bonus points for significant submissions.
