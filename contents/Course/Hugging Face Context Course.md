## Summary

The **Hugging Face Context Course** is a free, hands-on course focused on machine learning with agent context. It teaches how to train models, optimize inference, and build datasets using **SKILLS.md, Plugins, MCP (Model Context Protocol), Subagents, and Hooks**.

Key features:
- **Projects**: Includes pre-training, post-training (fine-tuning), and inference optimization (using llama.cpp).
- **Interactive Quizzes**: Each unit has quizzes to test understanding and earn certificates.
- **Supported Agents**: Works with **Pi, Claude Code, Codex, OpenCode** (instructions provided). Concepts also apply to Cursor or GitHub Copilot.
- **Prerequisites**: Basic Python, command line knowledge, Hugging Face account, and a code agent (Claude Code, Codex, or OpenCode).
- **Community & Contributions**: Open-source course; contributions via GitHub issues/PRs are welcome. Weekly live AMA sessions (first on May 12th).
- **Resources**: Access to quizzes, Spaces (e.g., Hook Lifecycle, Subagent Patterns), models, and datasets.

Enrollment is free via the Hugging Face organization.
