## Summary

The content outlines a course on prompt evaluations provided by Anthropic, consisting of nine lessons designed to help users implement evaluations using the Anthropic API. Key topics include:

1. **Evaluations 101** - Introduction to the fundamentals of evaluations.
2. **Human-graded evaluations** - Utilizing Anthropic's Workbench for creating human-graded evaluations.
3. **Code-graded evaluations** - Developing simple code-graded evaluations.
4. **Classification evaluations** - Writing and implementing classification evaluations.
5. **Promptfoo introduction** - An overview of using Promptfoo for evaluations.
6. **Classification evaluations with Promptfoo** - Detailed guidance on writing classification evaluations using Promptfoo.
7. **Custom graders** - Creating custom grading systems with Promptfoo.
8. **Model-graded evaluations** - Implementing model-graded evaluations using Promptfoo.
9. **Custom model-graded evaluations** - Advanced techniques for custom model-graded evaluations.

The course is structured to build upon concepts progressively, making it essential to follow the lessons in order.
