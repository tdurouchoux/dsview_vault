## Summary

The notes provide an introductory overview of machine learning with large-scale graphs, aligned with Stanford's CS224W course. They are structured into three main areas:

1. **Preliminaries**: Covers foundational concepts such as graph structure, network properties, and motifs.
2. **Network Methods**: Discusses various techniques including structural roles, spectral clustering, influence maximization, outbreak detection, link analysis, network robustness, and evolution, as well as knowledge graphs.
3. **Machine Learning with Networks**: Focuses on methods like message passing, node classification, representation learning (including techniques like DeepWalk and node2vec), graph neural networks (GCN, SAGE, GAT), and generative models for graphs (GraphRNN).

The notes are still being developed, and contributions for improvements are encouraged via GitHub.
